# Prism

是一款轻量级、高性能的 MCP (Model Context Protocol) 协议转换网关。它旨在将企业内部现有的服务能力（Swagger）无缝转化为 AI
Agent 可调用的工具（Tools），并针对私有化 AI 平台提供深度定制能力。

# 🌟 项目初心 (Vision)

在 AI Agent 落地企业的过程中，我们发现开发者面临着三个巨大的鸿沟：

协议鸿沟：企业内部大量的 RESTful/RPC 接口无法被 AI 直接理解，手动编写 MCP Server 重复劳动多、维护成本高。

管理鸿沟：成千上万的微服务接口，哪些该给 AI 用？如何像搭积木一样自由组合工具集？

适配鸿沟：私有化内网 AI 平台往往拥有非标的鉴权和通信逻辑，标准网关难以灵活适配。

Prism 的出现，就是为了做那个“最懂业务”的神经突触。它不追求做一个笨重的全量网关，而是专注于自动化转换、精细化过滤、以及极致的自定义适配

# 🛠️ 技术栈 (Tech Stack)

我们选择了 Python 生态，以确保在 AI 爆发式发展的今天，项目拥有最强的灵活性和最快的交付速度：

核心框架: FastAPI - 高性能异步 Web 框架，负责管理后台及非标协议对接。

协议转换: FastMCP - 强大的 MCP 协议组件，负责 OpenAPI 到底层工具的映射。

服务发现: Nacos Python SDK - 实时监听微服务状态，实现接口的自动感知。

异步通信: Uvicorn + [uvloop] - 支撑高并发长连接（SSE）的生产级服务器。

配置中心: Redis / YAML - 存储工具分类定义及自由组合配置。

# ✨ 核心特性

Nacos 自动联动：一键从 Nacos 发现服务，实时拉取 Swagger 定义。

工具自由拼装：支持在 UI/配置中自由挑选 API，并将其归类为逻辑上的“工具箱”。

非标鉴权适配：提供插件式拦截器，轻松搞定内网 AI 平台奇特的 Header 签名与握手逻辑。

轻量化部署：支持水平扩展，不仅能跑在本地，更能扛住生产级长连接压力。

安全沙箱：内置路径过滤白名单，确保只有“无副作用”的接口暴露给 AI。
